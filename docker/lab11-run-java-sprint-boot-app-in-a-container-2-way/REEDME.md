### ğŸ”¹ Step 3: Create the Dockerfile

Create a file named `Dockerfile` in the root of your project and add:

```dockerfile

WORKDIR /app
COPY target/demo-0.0.1-SNAPSHOT.jar app.jar
EXPOSE 8080
CMD ["java", "-jar", "app.jar"]
```
